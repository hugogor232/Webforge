Backend & Data

Propulsé par Supabase

Nous avons choisi Supabase pour offrir une expérience utilisateur fluide, sécurisée et temps réel.

  • PostgreSQL Natif

    Vos données de projets sont stockées dans une base de données relationnelle robuste et performante.

  • Realtime Subscriptions

    Le Dashboard se met à jour instantanément via WebSockets lors de la génération du site.

  • Authentification Unifiée

    Gestion sécurisée des sessions via Email/Password et OAuth (Google, GitHub).

supabaseClient.js 

import { createClient } from '@supabase/supabase-js'

const supabase = createClient(URL, KEY)

// Realtime subscription
const channel = supabase
  .channel('project_status')
  .on('postgres_changes', 
    { event: 'UPDATE', schema: 'public' }, 
    (payload) => updateUI(payload)
  )
  .subscribe()

Sécurité

Vos données sont isolées

La sécurité n'est pas une option. Nous utilisons les fonctionnalités avancées de PostgreSQL pour garantir la confidentialité.

Row Level Security (RLS)

Chaque requête vers la base de données est filtrée par le moteur PostgreSQL. Un utilisateur ne peut techniquement voir et modifier QUE les lignes qui lui appartiennent (basé sur son auth.uid()).

Architecture Serverless

Pas de serveurs à gérer, pas de failles d'OS non patchées. L'infrastructure évolue automatiquement avec la charge.
